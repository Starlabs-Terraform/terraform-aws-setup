# Terraform-aws-setup
### Terraform Backend S3 - Setup

> ### Terraform Setup
> ```
> git clone https://github.com/Starlabs-Terraform/terraform-aws-setup.git
> cd terraform-aws-setup
> terraform init
> terraform apply
> ```

> ### Opentofu Setup
> ```
> git clone https://github.com/Starlabs-Terraform/terraform-aws-setup.git
> cd terraform-aws-setup
> tofu init
> tofu apply
>```
